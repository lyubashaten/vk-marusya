# vk-marusya
«VK Маруся» — это бесплатная платформа для поиска и оценки фильмов. 🔍 Ищите фильмы по названию, актёру или году ⭐ Узнавайте рейтинг, читайте описание и смотрите трейлеры ❤️ Сохраняйте понравившееся в избранное и возвращайтесь к ним позже  Платформа подойдёт всем, кто хочет вести свой кинодневник и не забывать, что посмотреть вечером.
